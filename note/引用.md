# 引用
> 给变量其别名

## 基本使用

语法 ： 数据类型 & 别名 == 原名

~~~ cpp
int a=10;
int & b =a;
std::cout << a << std::endl; // a b 指向同一块内存
~~~

## 注意事项

1. 必须初始化
2. 引用初始化后，不可改变

## 引用做函数参数

作用： 简化指针（让形参变成实参）

~~~ cpp
void swap(int & a,int & b){
    int c=a;
    a=b;
    b=c;
}
~~~

##  引用做函数返回值

1. 不要返回局部变量 >他空间释放了 //现在是可以了好像
2. 函数调用可以修改左值
